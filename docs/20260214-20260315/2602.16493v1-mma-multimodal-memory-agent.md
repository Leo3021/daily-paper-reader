## 摘要
长程多模态智能体依赖于外部记忆；然而，基于相似性的检索往往会提取出陈旧、低可信度或冲突的项，这可能引发过度自信的错误。我们提出了多模态记忆智能体（MMA），它通过结合来源可信度、时间衰减和冲突感知的网络共识，为每个检索到的记忆项分配动态可靠性评分，并利用该信号重新加权证据，在支持不足时选择弃权。我们还引入了 MMA-Bench，这是一个通过程序生成的基准测试，用于评估具有受控说话者可靠性和结构化文本-视觉矛盾的信念动力学。利用该框架，我们揭示了“视觉安慰剂效应”（Visual Placebo Effect），揭示了基于检索增强生成（RAG）的智能体如何从基础模型中继承潜在的视觉偏见。在 FEVER 数据集上，MMA 在保持与基准模型相当的准确率的同时，将方差降低了 35.2% 并提高了选择性效用；在 LoCoMo 上，面向安全的配置提高了可执行准确率并减少了错误回答；在 MMA-Bench 上，MMA 在视觉模式下达到了 41.18% 的 Type-B 准确率，而基准模型在相同协议下则崩溃至 0.0%。代码：https://github.com/AIGeeksGroup/MMA。

## Abstract
Long-horizon multimodal agents depend on external memory; however, similarity-based retrieval often surfaces stale, low-credibility, or conflicting items, which can trigger overconfident errors. We propose Multimodal Memory Agent (MMA), which assigns each retrieved memory item a dynamic reliability score by combining source credibility, temporal decay, and conflict-aware network consensus, and uses this signal to reweight evidence and abstain when support is insufficient. We also introduce MMA-Bench, a programmatically generated benchmark for belief dynamics with controlled speaker reliability and structured text-vision contradictions. Using this framework, we uncover the "Visual Placebo Effect", revealing how RAG-based agents inherit latent visual biases from foundation models. On FEVER, MMA matches baseline accuracy while reducing variance by 35.2% and improving selective utility; on LoCoMo, a safety-oriented configuration improves actionable accuracy and reduces wrong answers; on MMA-Bench, MMA reaches 41.18% Type-B accuracy in Vision mode, while the baseline collapses to 0.0% under the same protocol. **研究动机与背景：**
在长程交互场景中，多模态智能体（Agents）高度依赖外部记忆系统来维持上下文。然而，现有的记忆增强架构（如基于 RAG 的系统）存在一个核心缺陷：它们通常假设检索到的所有记忆项都是同等可靠的。在实际应用中，检索到的信息可能包含陈旧数据、低可信度来源或相互矛盾的内容。这导致智能体在推理时容易受到噪声干扰，产生“幻觉”或过度自信的错误回答。

如何在存在噪声、过时信息和多模态冲突的情况下，提升智能体对记忆项可靠性的评估能力，并使其在证据不足时能够表现出“认知谨慎”（即主动弃权）？

## 2. 方法论
MMA 提出了一种在推理阶段运行的**元认知可靠性层**，对检索到的记忆进行动态加权和过滤。

### 核心思想
为每个检索到的记忆项 $M_i$ 计算一个综合置信度评分 $C(M_i)$，该评分由三个维度加权构成：
1.  **来源可靠性 (Source Reliability, S)**：根据记忆来源（如特定用户或数据库）预设的信任先验进行映射。
2.  **时间衰减 (Temporal Decay, T)**：采用指数衰减模型 $T(M_i) = \exp(-\frac{\ln(2)}{T_{half}} \Delta t_i)$，模拟信息随时间的老化。
3.  **网络共识 (Network Consensus, $C_{con}$)**：通过计算语义邻域内的支持度来衡量一致性。公式考虑了邻近记忆项的相似度加权，正值强化置信度，负值（矛盾）则惩罚置信度。

### 算法流程
*   **检索阶段**：从外部存储中提取相关记忆。
*   **评分阶段**：计算上述三个维度的得分，并进行自归一化加权求和。
*   **推理阶段**：高置信度记忆被优先用于推理；若整体证据置信度低于阈值，智能体触发弃权机制（Abstain）。

## 6. 主要结论与发现
1.  **稳定性显著提升**：在 FEVER 任务中，MMA 将性能波动的方差降低了 **35.2%**。
2.  **视觉安慰剂效应 (Visual Placebo Effect)**：研究发现，仅仅提供视觉图像（即使是模糊或无关的）就会显著降低智能体的弃权意愿，使其产生虚假的确定性。MMA 有效缓解了这一偏见。
3.  **解决认知瘫痪**：在极端噪声下，基准模型往往因为无法处理冲突而“瘫痪”（准确率为 0%），MMA 通过可靠性过滤恢复了决策能动性（在 MMA-Bench 视觉模式下达到 41.18% 准确率）。
4.  **安全-效用平衡**：MMA 在安全敏感场景下表现出更好的选择性预测能力，减少了错误回答的数量。

## 7. 优点
*   **多维度可靠性建模**：将来源、时间和语义共识结合，比单纯的相似度检索更符合人类的认知逻辑。
*   **MMA-Bench 的诊断价值**：该基准能精准识别智能体是“由于检索不到而弃权”还是“由于发现冲突而弃权”。
*   **即插即用**：作为后处理模块，不需要重新训练基础模型，具有较强的通用性。

---

## 8. 不足与局限
*   **检索召回率依赖**：MMA 属于“后过滤”机制。如果底层的 RAG 系统未能检索到关键证据，MMA 无法修复缺失的信息。
*   **稀疏性挑战**：在信息密度极低的场景下，严格的“共识检查”可能会导致智能体过于保守，从而降低召回率。
*   **计算开销**：虽然不涉及模型训练，但在推理时计算多个记忆项之间的两两相似度（共识模块）会增加一定的延迟。
