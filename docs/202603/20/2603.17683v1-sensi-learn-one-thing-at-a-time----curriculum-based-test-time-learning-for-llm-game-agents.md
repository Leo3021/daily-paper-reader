## 摘要
部署在未知环境中的大语言模型（LLM）智能体必须在测试时学习任务结构，但目前的方法需要数千次交互才能形成有用的假设。我们提出了 Sensi，这是一种针对 ARC-AGI-3 游戏挑战的 LLM 智能体架构，它通过三种机制引入了结构化的测试时学习：(1) 一种将感知与动作分离的双玩家架构；(2) 一个由外部状态机管理的基于课程的学习系统；(3) 一种将数据库作为控制平面的设计，使智能体的上下文窗口可通过编程方式进行引导。我们还引入了一个带有动态生成评估细则的 LLM-as-judge 组件，用于确定智能体何时对某一主题学习充分，从而进入下一阶段。我们报告了两个迭代版本的结果：Sensi v1 仅使用双玩家架构解决了 2 个游戏关卡，而 Sensi v2 增加了课程学习，虽然解决了 0 个关卡，但在约 32 次动作尝试中完成了整个学习课程，其样本效率比需要 1600-3000 次尝试的同类系统高出 50-94 倍。我们精确地将失败模式诊断为源自感知层的自洽幻觉级联，这表明架构瓶颈已从学习效率转向感知落地（perceptual grounding）——这是一个更易处理的问题。

## Abstract
Large language model (LLM) agents deployed in unknown environments must learn task structure at test time, but current approaches require thousands of interactions to form useful hypotheses. We present Sensi, an LLM agent architecture for the ARC-AGI-3 game-playing challenge that introduces structured test-time learning through three mechanisms: (1) a two-player architecture separating perception from action, (2) a curriculum-based learning system managed by an external state machine, and (3) a database-as-control-plane that makes the agents context window programmatically steerable. We further introduce an LLM-as-judge component with dynamically generated evaluation rubrics to determine when the agent has learned enough about one topic to advance to the next. We report results across two iterations: Sensi v1 solves 2 game levels using the two-player architecture alone, while Sensi v2 adds curriculum learning and solves 0 levels - but completes its entire learning curriculum in approximately 32 action attempts, achieving 50-94x greater sample efficiency than comparable systems that require 1600-3000 attempts. We precisely diagnose the failure mode as a self-consistent hallucination cascade originating in the perception layer, demonstrating that the architectural bottleneck has shifted from learning efficiency to perceptual grounding - a more tractable problem.

*   **核心问题**：当前的 LLM 智能体在面对完全未知的环境（如 ARC-AGI-3 游戏挑战）时，测试时学习（Test-Time Learning）的效率极低。现有系统（如 Symbolica 的 Agentica）通常需要 1,600 到 3,000 次交互才能建立起对游戏机制的初步理解。
*   **研究动机**：在计算预算受限的实际应用中，这种低效性限制了 LLM 智能体的实用性。作者旨在通过结构化的学习方法，显著提升智能体在未知环境中发现规则和逻辑的样本效率（Sample Efficiency）。

#### 2. 方法论
Sensi 架构经历了两个版本的迭代，核心思想是“感知与动作分离”以及“一次只学一件事”。
*   **双玩家架构 (Sensi v1)**：将智能体拆分为两个角色：
    *   **观察者 (Observer)**：负责维护关于游戏世界的假设列表（Guesses）和已确认事实（Figured-out）。
    *   **行动者 (Actor)**：根据观察者的假设选择动作，以验证假设或执行任务。
*   **课程学习系统 (Sensi v2)**：在 v1 基础上引入了结构化学习流程：
    *   **外部状态机**：管理学习目标的顺序（如：先学动作效果，再学能量机制，最后学获胜条件）。状态包括：未开始、学习中、已完成。
    *   **数据库即控制平面 (Database-as-control-plane)**：使用 SQLite 存储智能体的所有认知状态（事实、假设、历史）。每轮对话前，系统从数据库中提取相关信息注入 Prompt，使 LLM 的上下文窗口可编程且可控。
    *   **感知评分器 (Sense Scorer)**：引入 LLM-as-judge 机制。首先由一个 LLM 生成评估细则（Rubric），再由另一个 LLM 根据细则对当前学习进度打分（1-10分）。当分数达到阈值（如 8 分）时，状态机切换到下一个学习目标。
*   **感知层 (Frame Differencing)**：使用多模态 LLM 对相邻两帧图像进行差异分析，生成结构化的 JSON 描述，辅助观察者理解变化。

#### 6. 主要结论与发现
*   **样本效率飞跃**：Sensi v2 在约 **32 次** 交互内完成了全部学习课程，比基准系统（1,600+ 次）提升了 **50-94 倍** 的样本效率。
*   **瓶颈转移**：研究发现，限制智能体性能的瓶颈已从“学习逻辑的效率”转移到了“感知的准确性”。
*   **自洽幻觉级联 (Self-consistent Hallucination Cascade)**：这是 v2 失败的核心原因。感知层（帧差异分析）产生的微小错误会被后续的假设生成和评分机制放大。由于错误的信息在逻辑上是自洽的，评分器会给出高分，导致智能体系统性地“学会了错误的事实”。

#### 7. 优点
*   **架构创新**：提出的“数据库即控制平面”模式为长程任务中的状态管理提供了新思路，增强了智能体的可调试性和可引导性。
*   **效率极高**：通过课程学习显著降低了探索未知环境所需的交互次数。
*   **诊断精准**：对失败模式（幻觉级联）的深入分析为后续改进感知落地（Perceptual Grounding）指明了方向。

#### 8. 不足与局限
*   **感知脆弱**：完全依赖 LLM 进行视觉差异分析容易产生幻觉，缺乏底层像素级的硬约束。
*   **课程依赖**：目前的学习课程（学习项及其顺序）是人工设计的，尚未实现自动生成课程。
*   **任务成功率**：v2 版本虽然学得快，但由于感知错误，未能转化为实际的通关能力。
*   **模型一致性**：对比实验中更换了底层模型，使得架构改进带来的收益与模型能力差异难以完全解耦。
