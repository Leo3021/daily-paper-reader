## 摘要
将大语言模型（LLMs）与人类价值观对齐是确保其可靠和安全部署的核心挑战。虽然现有方法（如来自人类反馈的强化学习 RLHF 及其变体）已经改善了对齐效果，但它们通常依赖于单一评估者或定义狭窄的奖励信号，限制了其捕捉伦理多元主义的能力。在这项工作中，我们提出了基于组合融合分析的价值对齐系统（VAS-CFA），这是一个实现多智能体融合对齐的框架。该系统实例化了多个道德智能体，每个智能体都经过微调以代表不同的规范视角，并使用 CFA 通过基于排名和分数的聚合来融合它们的输出。这种设计利用了智能体之间的认知多样性，以减轻多个智能体之间的冲突和冗余，从而产生更好地反映人类价值观的响应。实证评估表明，VAS-CFA 在标准指标上优于单智能体基准和先前的聚合方法，表明多智能体融合为推进大语言模型的价值对齐提供了一种稳健且有效的机制。

## Abstract
Aligning large language models (LLMs) with human values is a central challenge for ensuring trustworthy and safe deployment. While existing methods such as Reinforcement Learning from Human Feedback (RLHF) and its variants have improved alignment, they often rely on a single evaluator or narrowly defined reward signals, limiting their ability to capture ethical pluralism. In this work, we propose the Value Alignment System using Combinatorial Fusion Analysis (VAS-CFA), a framework that operationalizes multi-agent fusion alignment. It instantiates multiple moral agents, each fine-tuned to represent a distinct normative perspective, and fuses their outputs using CFA with both rank- and score-based aggregation. This design leverages cognitive diversity, between agents, to mitigate conflicts and redundancies across multiple agents, producing responses that better reflect human values. Empirical evaluation demonstrates that VAS-CFA outperforms both single agent baselines and prior aggregation approaches on standard metrics, showing that multi-agent fusion provides a robust and effective mechanism for advancing value alignment in LLMs.

*   **核心问题**：如何通过多智能体协作和数学融合框架，使 LLM 的输出能够同时兼顾多种道德维度（如公平、关怀、权威等），从而实现更稳健、更符合人类多元价值观的对齐。

该论文提出了 **VAS-CFA（基于组合融合分析的价值对齐系统）**，其核心流程如下：
*   **多道德智能体实例化**：基于 `Pythia-12B` 模型，利用 **DPO（直接偏好优化）** 和 **QLoRA** 技术，在道德诚信语料库（MIC）上分别微调出 5 个专门的智能体，每个智能体代表一种道德维度：权威（Authority）、关怀（Care）、公平（Fairness）、忠诚（Loyalty）和圣洁（Sanctity）。

*   **道德评分与分类**：训练一个基于 SentenceTransformer 和逻辑回归的分类器，为每个道德单元在 5 个维度上进行评分。
*   **组合融合分析（CFA）**：
    *   **核心思想**：利用“认知多样性”（Cognitive Diversity）来衡量不同评分系统间的差异。
    *   **融合技术**：采用四种聚合方式：平均分融合（ASC）、基于多样性强度的加权分融合（WSCDS）、平均排名融合（ARC）以及加权排名融合（WRCDS）。
*   **改写输出**：根据 CFA 选出的最优道德单元，通过改写器（Paraphraser）生成最终的完整回答。

### 6. 主要结论与发现
*   **VAS-CFA 表现优异**：在 ROUGE-L 和 BERTScore 指标上，VAS-CFA 均显著优于所有单智能体基准和先前的多智能体聚合方法。

*   **多样性是关键**：利用智能体之间的差异性（而非仅仅是共性）能产生更符合人类复杂价值观的响应。

*   **理论扎实**：引入了成熟的组合融合分析（CFA）数学框架，为多模型融合提供了比简单平均更科学的解释。
*   **解决语义冲突**：通过将响应分解为“道德单元”再进行融合，有效解决了多智能体输出直接拼接时可能出现的逻辑矛盾或语义稀释问题。
*   **关注多元化**：突破了单一奖励模型的局限，为实现“伦理多元化”的 AI 对齐提供了可行路径。
